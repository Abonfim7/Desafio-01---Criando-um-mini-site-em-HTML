# Desafio-01---Criando-um-mini-site-em-HTML

	
Desafio 01 - Criando um mini site em HTML
🌐 Desafio: Criando um Mini Site em HTML  

Objetivo:
Criar um pequeno site utilizando os elementos que aprendemos em HTML: título, parágrafo, imagem, listas e navegação entre páginas.

--------

📋Instruções:
Você deverá criar um mini site com 3 páginas HTML conectadas entre si.
Os arquivos devem se chamar:
index.html
lugares.html
comidas.html
Todas as páginas devem ter links para navegar entre elas.

--------

🏠 Página 1 – Página Inicial (index.html)
A página inicial deve conter:
Um título principal
Um parágrafo falando sobre a cidade escolhida
Uma imagem da cidade ou de um ponto turístico
Um menu de navegação com links para as outras páginas
Exemplo de menu:
Início
Lugares para visitar
Comidas típicas
---------

📍 Página 2 – Lugares para Visitar (lugares.html)
Essa página deve conter:
Um título
Umparágrafo curto
Uma lista não ordenada com pelo menos 4 lugares turísticos
Exemplo de conteúdo:
Cristo Redentor
Pão de Açúcar
Praia de Copacabana
Maracanã
----------

🍛 Página 3 – Comidas Típicas (comidas.html)
Essa página deve conter:
Um título
Umparágrafo curto
Uma lista ordenada com pelo menos 4 comidas típicas
Exemplo:
Feijoada
Pão de queijo
Brigadeiro
Açaí
Brigadeiro
Açaí
